--Rendering Setup Toolkit guidelines.

In order to set the options, select the desired option in the popup menu, and click "set".

By default, the Main camera is selected under the panel "Camera settings", but you can select the one you need (if you have more than one camera).

--There's a known bug with lighting maps, where sometimes in dynamic objects, light appears to be broken. It can be fixed just by clicking "Generate Lighting" under Unity's Lighting settings. 
However we are working to solve this.

*This project is in it's 1.2 version, and it's being polished. Expect some bugs. Feel free to report them at the forum page for the product,
and maybe you want to add some functionality. Write some feedback too, we'd love to improve the tool!
